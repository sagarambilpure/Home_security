-------HOME SECURITY------
This project uses raspberry pi; when motion is detected buzzer makes sound and raspi clicks photo
and mails that photo to given users emailid

------Requirements----
->raspberry pi
->pir motion detector
->pi camera
->buzzer
->connecting wires

------procedure----
->connect pi-camera,buzzer(pin-24),pir motion detector(pin-23) to raspberry pi
->run motion_detect.py

-----Note----
Ground must be connected accordingly
